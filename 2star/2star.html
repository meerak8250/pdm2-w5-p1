Instructions

There are a lot of built-in events we can use to improve our program, in this exercise we will explore a few of them. In sketch.js, write a sketch where the canvas is always 50% of the width of the browser window. The height of the canvas should be the same as its width and the canvas should resize when the browser window resizes. Try to go through the process of exploring the p5.js documentation to find the right event function, but if you're not sure use the hint.

Add a character in the centre of the canvas. The character should stay in the centre when the canvas resizes AND it should grow or shrink proportionally. For example, if the canvas gets wider, the creature should get wider.

Example output:

To create a canvas that is 50% of the width of the browser window, you will need to do a little maths with the windowWidth system variable.

The following will help with canvas resizing:

  • windowResized() - an event handler that fires when the browser window is resized.
  • resizeCanvas() - resizes the canvas after it has been created.
